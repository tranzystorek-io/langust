# Langust

## Description

A list-processing language for the
*TKOM* (Compilation techniques) course at WUT.

## Usage

### Prerequisites

* cmake

### Build

* `mkdir build && cd build`
* `cmake ..`
* `make`

### Run

* Path to executable is `<build-dir>/repl/replangust`
* Type in commands
* To exit, send *EOF* (Ctrl+D on *nix)
